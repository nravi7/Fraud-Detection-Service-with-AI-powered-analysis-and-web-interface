API Endpoints

Once deployed, your service will have these endpoints:

  • GET / - Web interface
  • GET /health - Health check
  • POST /v1/score - OpenAI analysis
  • POST /v1/score/hf - Rule-based analysis
  • GET /docs - API documentation
